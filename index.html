系统能力

不是十八个一次性作品 —— 一套系统,换肤而成。

每套模板共享同一套 CSS 令牌层、同一套板块语汇、同一条无障碍底线。改的是配色,不是结构。

品牌色驱动

一键换成任意品牌色

一套 :root 令牌驱动全站配色。注入客户的品牌色板,整站即刻换肤 —— 无需逐处替换。

双主题

明暗都用心设计

响应系统 深色偏好 并支持手动切换;暗色主题是专门设计的,不是简单反色。

无障碍

构建即达标

标题层级顺序、可见文字即无障碍名 —— 18 套全部通过真机 Lighthouse 同款校验。

性能

真机 99 分

真实手机节流环境实测:LCP 1.7s、CLS 0、TBT 0ms。自包含标记,无阻塞渲染资源。

标准

基于 Dawn 15.4.1

板块结构对齐 Shopify 官方参考主题,每个模块都能落到平台已识别的结构上。

自包含

零外部请求

CSS 全内联、图形用 CSS 渲染、字体走系统栈。无 CDN、无外链字体、无第三方调用。

forge 如何选用

接单信息定蓝图,品牌色完成适配。

模板不是一份让你复制的起始文件 —— 它由品牌接单信息自动选出,再按品牌色板换肤。

接单信息自动定蓝图

行业 × 商业模式 × 页面类型经注册表映射到模板 —— B2B 走批发版式,冷门行业回落到最接近的一套。

页面计划带上选择

page-plan.yaml 每一页都会带上 baseline_template —— 构建方清楚该从哪套蓝图起步。

品牌色完成适配

select-template.js --adapt 把品牌 design-tokens.json 注入模板配色。版式不变,换成客户的颜色。

artifacts/lumina-beauty/page-plan.yaml
# 计划阶段自动解析
pages:
  - type: home
    baseline_template: home-dtc-beauty.html
  - type: product
    baseline_template: product-beauty.html
  - type: collection
    baseline_template: collection-beauty.html
