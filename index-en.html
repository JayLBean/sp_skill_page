Capabilities

Not eighteen one-off builds — one system, reskinned.

Every template shares the same CSS token layer, the same section vocabulary, the same accessibility baseline. What changes is the color, not the structure.

BRAND-COLOR DRIVEN

Swap in any brand color with one click

A single set of :root tokens drives the entire site's colors. Inject your client's brand palette and the whole site reskins instantly — no need to replace anything piece by piece.

DUAL THEME

Both light and dark, thoughtfully designed

Responds to the system dark-mode preference and supports a manual toggle; the dark theme is purpose-designed, not a simple inversion.

ACCESSIBILITY

Compliant by construction

Ordered heading hierarchy, visible text as accessible name — all 18 templates pass the same real-device Lighthouse checks.

PERFORMANCE

99 on real devices

Measured on a real, throttled phone: LCP 1.7s, CLS 0, TBT 0ms. Self-contained markup, no render-blocking resources.

STANDARD

Built on Dawn 15.4.1

Section structure aligns with Shopify's official reference theme, so every module maps onto structures the platform already recognizes.

SELF-CONTAINED

Zero external requests

All CSS inlined, graphics rendered with CSS, fonts on the system stack. No CDN, no external fonts, no third-party calls.

How forge picks a template

The intake brief sets the blueprint; the brand color completes the fit.

A template isn't a starter file for you to copy — it's selected automatically from the brand intake brief, then reskinned to the brand palette.

The intake brief sets the blueprint automatically

Industry × business model × page type is mapped to a template through a registry — B2B uses the wholesale layout, and niche industries fall back to the closest match.

The page plan carries the selection

Every page in page-plan.yaml carries a baseline_template — the builder knows exactly which blueprint to start from.

The brand color completes the fit

select-template.js --adapt injects the brand design-tokens.json into the template's colors. The layout stays the same, recolored to the client's palette.

artifacts/lumina-beauty/page-plan.yaml
# resolved automatically during the plan stage
pages:
  - type: home
    baseline_template: home-dtc-beauty.html
  - type: product
    baseline_template: product-beauty.html
  - type: collection
    baseline_template: collection-beauty.html
